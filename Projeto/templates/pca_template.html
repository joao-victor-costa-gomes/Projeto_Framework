PRINCIPAL COMPONENT ANALYSIS - PCA

PCA significa Análise de Componentes Principais (Principal Component Analysis, em inglês). É uma técnica estatística utilizada para simplificar a complexidade em conjuntos de dados de alta dimensão, reduzindo o número de variáveis ​​ao mesmo tempo em que preserva a maior quantidade possível de informações. O PCA realiza essa redução de dimensionalidade transformando os dados originais em um novo conjunto de variáveis não correlacionadas chamadas de componentes principais. Esses componentes principais são ordenados de forma que o primeiro capture a maior quantidade possível de variância nos dados, o segundo capture a segunda maior quantidade de variância e assim por diante. PCA é amplamente utilizado em várias áreas, incluindo análise de dados, reconhecimento de padrões, visão computacional, entre outros.


GRÁFICO DO PCA

PCA reduz a dimensionalidade dos dados projetando-os em um novo espaço de menor dimensão, enquanto tenta preservar a maior quantidade possível de variância dos dados originais. Quando você reduz os dados para duas dimensões usando PCA, está essencialmente encontrando um novo conjunto de eixos ortogonais (ou componentes principais) nos quais os dados podem ser representados de forma mais compacta.
